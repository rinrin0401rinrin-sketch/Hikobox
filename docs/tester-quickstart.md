# 共有テスト版の使い方

## 共有者向けの最短手順
1. 受け取った URL を iPhone の Safari で開く
2. 画面上部の `テスト版` 表示が案内された版になっているか確認する
3. `小選挙区` `比例代表` `検索` を順に開く
4. `検索` でひらがな 2 文字以上を入れて `検索する` を押す
5. `めくる` を押して裏面が表示されるか確認する

## PWA として試す手順
1. Safari の共有ボタンを押す
2. `ホーム画面に追加` を選ぶ
3. 追加後はホーム画面のアイコンから起動する
4. Safari 版と同じく `テスト版` 表示を確認する

## 不具合報告で伝えてほしいこと
- どのタブで起きたか
- 何を入力したか
- 画面上部の `テスト版` 表示が何になっていたか
- Safari で起きたか、ホーム画面追加版で起きたか

## 反映されない時
1. Safari のタブを閉じる
2. URL を開き直す
3. それでも古いままなら、Safari のサイトデータを削除して再読み込みする
