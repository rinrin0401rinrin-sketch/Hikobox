# 共有 URL 最終チェック

## 送る前に見る項目
- URL をブラウザで開いて表示できる
- 画面上部の `テスト版` 表示が案内した版と一致する
- `小選挙区` `比例代表` `検索` の3タブが開く
- `検索` でひらがな検索が通る
- カード表面が顔写真のみ、裏面で情報が見える
- Safari で `ホーム画面に追加` の導線が見える

## 共有相手に伝える項目
- Safari 推奨
- 画面上部の `テスト版` 表示を確認してほしい
- 不具合時はタブ名、入力内容、テスト版表示、Safari / ホーム画面版の別を教えてほしい

## 反映されない時の案内
1. タブを閉じる
2. URL を開き直す
3. それでも古い場合は Safari のサイトデータを削除する
