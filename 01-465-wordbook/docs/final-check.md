# 最終チェック

## 1. 公開前の基本確認
1. `npm run build:search-index`
2. `npm run build:thumbnails`
3. `npm run verify:release`
4. 失敗があれば修正して再実行する

## 2. 共有前の確認
- 共有 URL が開く
- 画面上部の `テスト版` 表示が案内した版と一致する
- `小選挙区` `比例代表` `検索` の3タブが開く
- `検索` でひらがな検索が通る
- カード表面が顔写真だけで、裏面に情報がある

## 3. iPhone 実機の最終確認
- Safari で URL が開く
- `ホーム画面に追加` ができる
- ホーム画面起動後に上端余白や固定タブが不自然でない
- `小選挙区` の都道府県が地域順になっている
- 写真の崩れ、タップしにくさ、スクロール詰まりがない
- `検索する` と `めくる` が自然に使える

## 4. 合格条件
- `npm run verify:release` が通る
- 実機で重大な崩れがない
- 顔写真と氏名の対応に問題がない
- 共有相手が `tester-quickstart` だけで触り始められる

## 5. 共有時に渡す文面
- Safari で URL を開いてください
- 上部の `テスト版` 表示を確認してください
- `小選挙区` `比例代表` `検索` を順に試してください
- 表面が顔写真だけ、`めくる` で裏面情報が出るか確認してください
- 不具合時はタブ名、入力内容、テスト版表示、Safari / ホーム画面版の別を教えてください
