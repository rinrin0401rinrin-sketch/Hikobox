# Markdown Novel Template

Use this structure when drafting a new novel or consolidating scattered notes into one manuscript.

```md
# Title

## Logline
- One or two sentences that state the hook, protagonist, and conflict.

## Premise
- Short explanation of the setup and why the story starts here.

## Story Bible

### Protagonist
- Name:
- Goal:
- Internal conflict:
- Signature traits:

### Main Cast
- Name / role / tension with protagonist

### Setting Rules
- World rules that materially affect plot decisions

### Ending Promise
- What emotional or plot resolution the novel is driving toward

## Chapter Plan

### Chapter 1: [Chapter Title]
- Opening hook
- Scene goal
- Turn or reveal
- Exit hook

### Chapter 2: [Chapter Title]
- Opening hook
- Scene goal
- Turn or reveal
- Exit hook

## Manuscript

## Chapter 1: [Chapter Title]
[Write the chapter in prose]

## Chapter 2: [Chapter Title]
[Write the chapter in prose]

## Revision Notes
- Continuity checks
- Pacing issues
- Open questions
```

## Usage Notes

- Keep planning and prose in the same Markdown file when the user wants one deliverable.
- Collapse sections the user does not need instead of inventing filler.
- If the user asks for a chapter-only draft, keep `Story Bible` short and spend tokens on the manuscript.
